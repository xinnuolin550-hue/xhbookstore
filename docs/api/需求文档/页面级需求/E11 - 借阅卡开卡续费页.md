#### 开卡步骤

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 填写会员信息 | 姓名、手机号等基本信息（或通过微信授权获取） |
| 2 | 选择卡类型 | 选择年卡/半年卡等，不同卡类型对应不同有效期和借阅额度 |
| 3 | 确认开卡 | 提交开卡申请，系统生成会员编号和会员卡 |
| 4 | 开卡成功 | 展示会员卡信息：卡号、有效期、借阅额度等 |

#### 卡类型参考

| 卡类型 | 有效期 | 借阅额度 | 说明 |
|--------|--------|----------|------|
| 普通年卡 | 365天 | 10本 | 标准借阅卡 |
| 普通半年卡 | 183天 | 5本 | 短期借阅卡 |
| 银卡年卡 | 365天 | 20本 | 升级卡种 |
| 金卡年卡 | 365天 | 30本 | 高级卡种 |

#### 续费步骤

### 3. 有效期计算规则

#### 新开卡

```
开卡时间: 2026-06-30 10:00:00
年卡有效期: 2026-06-30 10:00:00 ~ 2027-06-30 23:59:59 (365天)
半年卡有效期: 2026-06-30 10:00:00 ~ 2026-12-30 23:59:59 (183天)
```

#### 续费（在原有效期基础上延长）

```
原有效期: 2026-06-30 ~ 2027-06-30
续费年卡: 有效期延长至 2028-06-30 (在当前到期日基础上 +365天)
续费半年卡: 有效期延长至 2027-12-30 (在当前到期日基础上 +183天)
```

## 业务逻辑规划

1. **开卡**:
   - 校验手机号是否已注册
   - 根据卡类型生成会员编号（前缀 `650000000` + 4位序列）
   - 设置有效期：当前时间 + 卡类型对应的天数
   - 设置初始化借阅额度
   - 可选赠送新会员注册积分
2. **续费**:
   - 校验会员是否存在且卡状态正常
   - 在原有效期满的基础上延长对应天数（非覆盖）
   - 更新会员的 validDate 字段
   - 记录续费操作日志
3. **卡类型管理**:
   - 卡类型定义在 `card_type` 表中，包含名称、有效期天数、借阅额度、价格
   - 新增卡类型通过管理后台配置（xhbookstore-admin）
