# E01 — 员工首页

> 对应代码: `StaffController.java` | 基础路径: `/api/mp/v1/staff`

员工登录小程序后进入的首页，展示当前店铺名称、今日全店借阅量和今日个人办理借阅量，为员工提供快速业务概览。

---

## 关联接口

| 接口 | 方法 | 路径 | 认证 |
|------|------|------|------|
| 员工首页 | GET | `/api/mp/v1/staff/home` | 是 |

## 1. 员工首页

```
GET /api/mp/v1/staff/home
Authorization: Bearer <accessToken>
```

### 请求参数

无（员工身份从 Token 中解析）。

### 请求报文示例


| 字段 | 类型 | 说明 |

| data.todayStoreBorrowCount | Integer | 今日全店借阅量 |

## 业务逻辑

1. **身份识别**: 从 JWT Token 中解析员工所属门店(deptId)和员工ID。
2. **门店名称**: 根据员工所属门店查询门店信息，返回门店名称。
3. **今日全店借阅量**: 统计当天 00:00:00 至当前时间，本门店所有员工操作的借阅订单数量。
4. **今日个人办理量**: 统计当天 00:00:00 至当前时间，当前登录员工操作的借阅订单数量。
5. **缓存策略**: 数据实时查询，不做缓存，保证数据准确性。
