# E04 — 积分列表页

> 对应代码: `StaffController.java` | 基础路径: `/api/mp/v1/staff`

员工查看全市（本门店）会员积分变动记录，支持按手机号、会员ID、积分方向筛选，点击可查看积分详情。

---

## 关联接口

| 接口 | 方法 | 路径 | 认证 |
|------|------|------|------|
| 全市积分列表 | GET | `/api/mp/v1/staff/points-records` | 是 |
| 积分详情 | GET | `/api/mp/v1/staff/points-records/{pointsRecordId}` | 是 |

---

## 1. 全市积分列表

```
GET /api/mp/v1/staff/points-records
Authorization: Bearer <accessToken>
```

### 请求参数

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| phone | String | 否 | 会员手机号（模糊搜索） |
| memberId | String | 否 | 会员ID（精确匹配） |
| direction | String | 否 | 积分方向：`add`=增加、`deduct`=消耗 |
| pageNo | Integer | 否 | 页码，默认 1 |
| pageSize | Integer | 否 | 每页条数，默认 20 |

### 请求报文示例


## 业务逻辑

1. **全市积分列表**: 查询本门店（deptId）下所有会员的积分变动记录。
   - 支持按手机号模糊匹配会员
   - 支持按会员ID精确查询
   - 支持按积分方向筛选（增加/消耗）
2. **积分方向判断**: 订单号前缀 `IN` 表示增加、`OT` 表示消耗。
3. **积分详情**: 根据积分记录编号查询完整的积分变动信息。
4. **脱敏展示**: 手机号中间4位显示为 `****`。
