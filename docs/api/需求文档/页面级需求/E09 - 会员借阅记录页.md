# E09 — 会员借阅记录页

> 对应代码: `StaffController.java` | 基础路径: `/api/mp/v1/staff`

员工查看指定会员的借阅历史记录，支持按模式和状态筛选，展示会员概要信息和分页的借阅记录列表。

---

## 关联接口

| 接口 | 方法 | 路径 | 认证 |
|------|------|------|------|
| 会员借阅记录 | GET | `/api/mp/v1/staff/members/{memberId}/borrows` | 是 |

---

## 1. 会员借阅记录

```
GET /api/mp/v1/staff/members/{memberId}/borrows
Authorization: Bearer <accessToken>
```

### 请求参数

**路径参数**:

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| memberId | String | 是 | 会员ID |

**查询参数**:

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| mode | String | 否 | 查询模式：`all`=全部 `current`=当前借阅中（默认 all） |
| status | String | 否 | 借阅状态筛选 |
| pageNo | Integer | 否 | 页码，默认 1 |
| pageSize | Integer | 否 | 每页条数，默认 20 |

**借阅状态枚举 (borrowStatus)**:

| 值 | 说明 |
|----|------|
| 0 | 借阅中 |
| 1 | 部分归还 |
| 2 | 已全部归还 |
| 5 | 已取消 |

### 请求报文示例

```bash
curl -X GET "https://api.xhbookstore.com/api/mp/v1/staff/members/1/borrows?mode=all&pageNo=1&pageSize=10" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiJ9.xxxxx"
```

```bash
# 查询当前借阅中记录
curl -X GET "https://api.xhbookstore.com/api/mp/v1/staff/members/1/borrows?mode=current&pageNo=1&pageSize=10" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiJ9.xxxxx"
```

## 业务逻辑

1. **会员验证**: 根据 memberId 查询会员信息，不存在则返回错误。
2. **查询模式**:
   - `mode=all`: 查询该会员所有借阅记录
   - `mode=current`: 仅查询借阅中(0)和部分归还(1)的记录
3. **状态筛选**: 支持按 borrowStatus 精确筛选。
4. **数据组装**: 每个借阅记录包含订单主信息和图书明细列表，明细中包含每本书的借阅数量和已还数量。
5. **分页查询**: 按借阅时间倒序排列，最新借阅在前。
