## 页面布局

| 区域 | 包含功能 | 说明 |
|------|----------|------|
| 个人资料 | 头像、昵称、性别 | 点击进入编辑 |
| 通知设置 | 到期提醒、积分提醒、活动提醒 | 开关切换 |
| 关于 | 版本号、用户协议、隐私政策 | 静态信息 |
| 账号管理 | 注销账号 | 跳转至注销须知页 |

## 业务逻辑

1. **个人资料管理**: 用户可修改昵称和头像，手机号不可修改（微信绑定）。
2. **通知偏好**: 借阅到期提醒默认开启，可在到期前 N 天推送服务通知。
3. **账号注销**: 入口放在设置页底部，点击后跳转 C03 注销须知页进行资格检查。
