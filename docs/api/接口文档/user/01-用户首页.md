# 用户首页

> **接口路径**: `GET /api/mp/v1/user/home`
> **认证要求**: 是 (Bearer Token)
> **Content-Type**: application/json;charset=UTF-8
> **对应代码**: `UserController.java` — `home()` 方法

## 接口说明

获取已登录用户的首页概览数据，包括会员信息、会员卡状态、当前积分、借阅数量等关键指标。

## 请求参数

无（无需请求体，用户信息从 Token 中解析）。

## 请求报文

```json
（无请求体）
```

## curl 测试

```bash
curl http://localhost:8091/api/mp/v1/user/home \
  -H "Authorization: Bearer <accessToken>"
```

## 响应参数

| 字段 | 类型 | 说明 |
|------|------|------|
| code | int | 状态码（0 表示成功） |
| message | String | 提示信息 |
| requestId | String | 请求追踪 ID |
| data.phoneDisplay | String | 脱敏手机号 |
| data.member | Object | 会员信息对象 |
| data.member.memberId | Integer | 会员 ID |
| data.member.memberNo | String | 会员编号 |
| data.member.memberName | String | 会员姓名 |
| data.member.phoneDisplay | String | 脱敏手机号 |
| data.member.card | Object | 会员卡信息 |
| data.member.card.memberLevel | Integer | 会员等级（0-100） |
| data.member.card.cardName | String | 卡名称（如"年卡"、"半年卡"） |
| data.member.card.cardStatus | String | 卡片状态（active / expired / disabled） |
| data.member.card.effectiveAt | Long | 生效时间（毫秒时间戳） |
| data.member.card.expiredAt | Long | 过期时间（毫秒时间戳） |
| data.member.card.remainingDays | Integer | 剩余天数 |
| data.member.currentPoints | Integer | 当前可用积分 |
| data.member.currentBorrowingCount | Integer | 当前借阅中数量 |
| data.member.yearBorrowCount | Integer | 年度累计借阅数量 |

## 业务逻辑

1. 根据请求头 `Authorization` 中的 Bearer Token 解析当前登录用户。
2. 查询会员基本信息及关联的最新会员卡。
3. 会员卡状态 `cardStatus` 取值：`active`（有效）、`expired`（已过期）、`disabled`（已停用）。
4. 若会员未办理会员卡，`card` 字段返回 `null`。
5. `remainingDays` 根据 `expiredAt` 与当前时间计算得出，不足一天按一天计。
6. 当前实现返回固定模拟数据，实际需对接数据库查询。
