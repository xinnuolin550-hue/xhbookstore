# 微信手机号登录

> **接口路径**: `POST /api/mp/v1/auth/wechat-phone-login`
> **认证要求**: 否（白名单接口，受 IP 限流限制，默认 5 次/分钟）
> **Content-Type**: application/json;charset=UTF-8
> **对应代码**: `AuthController.java`

## 接口说明

通过微信授权 code 换取手机号，完成自动登录或注册。已注册会员返回完整会员信息，未注册访客返回临时身份标识。

---

## 请求参数

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| code | String | 是 | 无 | 微信小程序端通过 `wx.login` 获取的临时授权码（有效期 5 分钟） |

## 请求报文

```json
{
  "code": "063mQn000w8h0F1QHC000EKb3o3mQn0C"
}
```

## curl 测试

```bash
curl -X POST http://localhost:8091/api/mp/v1/auth/wechat-phone-login \
  -H "Content-Type: application/json;charset=UTF-8" \
  -d '{
    "code": "063mQn000w8h0F1QHC000EKb3o3mQn0C"
  }'
```

## 响应参数

| 字段 | 类型 | 说明 |
|------|------|------|
| code | Integer | 0=成功 |
| message | String | 提示信息 |
| data.accessToken | String | 访问令牌（JWT），有效期 7200 秒 |
| data.refreshToken | String | 刷新令牌（JWT），有效期 2592000 秒（30天） |
| data.expiresIn | Long | accessToken 有效期（秒），固定 7200 |
| data.isStaff | Boolean | 是否为已注册会员（true=会员/false=访客） |
| data.userId | String | 会员 ID 或临时 UUID |
| data.memberId | Long | 会员 ID（isStaff=true 时有值），访客时为 null |
| requestId | String | 本次请求唯一标识 |

## 业务逻辑

1. 接收微信 `code`，调用微信服务接口换取手机号（开发环境未配置时使用 Mock 数据）
2. 根据手机号查询会员表：存在则 `isStaff=true`，`memberId` 取会员 ID；不存在则 `isStaff=false`，生成随机 UUID 作为临时 `userId`
3. 生成双 Token：`accessToken`（2 小时有效）+ `refreshToken`（30 天有效），均使用 HS256 签名
4. `accessToken` 中携带 `userId`、`isStaff`、`phone`、`memberId`、`type=access`
5. `refreshToken` 中携带 `userId`、`type=refresh`、唯一 `jti`
6. 登录接口受 IP 级别限流保护，默认 5 次/分钟
